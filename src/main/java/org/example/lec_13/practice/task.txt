Task 1

Task 2
Есть некоторый текст, его нужно записать в файл. Можно использовать одно и то же предложение, запишите его 1000 раз

Task 3
Прочитать файл из Task_2 и вывести на экран содержимое

Task 4

Затем это сообщение нужно прочитать из файла и вывести на экран

Task 5
Написать функцию, которая бы обходила весь каталог (включая вложенные папки и файлы) и печатала бы на экран структуру.
Это может быть папка с вашим проектом
